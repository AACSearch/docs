# Обработка естественного языка

Обработка естественного языка (NLP, Natural Language Processing) в AACSearch позволяет платформе понимать сложные поисковые запросы, учитывать синонимы, контекст и намерения пользователя. Этот раздел объясняет, как работает NLP в AACSearch и как его можно настроить.

## Возможности NLP в AACSearch

AACSearch использует передовые алгоритмы обработки естественного языка для улучшения качества поиска:

- **Понимание запросов**: Разбор сложных фраз и вопросов на естественном языке.
- **Синонимы и связанные термины**: Автоматическое распознавание синонимов и связанных понятий.
- **Исправление опечаток**: Обработка запросов с ошибками или неточностями.
- **Контекстный поиск**: Учет контекста запроса для более точных результатов.
- **Многоязычность**: Поддержка поиска на разных языках с учетом языковых особенностей.

## Настройка NLP

AACSearch предоставляет инструменты для настройки обработки естественного языка под ваши нужды:

1. **Управление синонимами**  
   Добавляйте пользовательские синонимы для улучшения поиска.

   ```json
   {
     "synonyms": [
       {
         "objectID": "synonym-1",
         "type": "synonym",
         "synonyms": ["телефон", "смартфон", "мобильный"]
       }
     ]
   }
   ```

2. **Настройка толерантности к опечаткам**  
   Укажите, насколько строго система должна обрабатывать опечатки.

   ```json
   {
     "typoTolerance": {
       "enabled": true,
       "minWordSizeForTypos": {
         "oneTypo": 4,
         "twoTypos": 8
       }
     }
   }
   ```

3. **Поддержка многоязычности**  
   Настройте индекс для работы с несколькими языками.
   ```json
   {
     "queryLanguages": ["ru", "en"],
     "indexLanguages": ["ru", "en"]
   }
   ```

## Интеграция с внешними NLP-инструментами

Если у вас есть собственные модели обработки естественного языка, вы можете интегрировать их с AACSearch через API для передачи дополнительных данных или весов для ранжирования.

- Используйте endpoint `/query-suggestions` для отправки предложений по запросам.
- Передавайте данные через веб-хуки для автоматического обновления синонимов или правил.

Для более детальной информации о настройке NLP обратитесь к [руководству по синонимам](../guides/search/synonyms.md) и [многоязычному поиску](../tutorials/multi-language.md).
